## 🎯 Обзор

Это **полнофункциональная демонстрация** веб-версии CryptoTrader с имитированными данными. Демо работает полностью в браузере без необходимости установки серверов PHP, Java или баз данных.

### Что Включено

#### 📊 Интерактивные Графики
- **TradingView Lightweight Charts** - профессиональная библиотека для трейдинга
- Свечные графики (Candlestick) с реалистичными данными
- 5 криптовалютных пар: BTC/USDT, ETH/USDT, BNB/USDT, SOL/USDT, ADA/USDT
- 6 временных интервалов: 1m, 5m, 15m, 1h, 4h, 1d
- Автоматическое обновление каждые 5 секунд

#### 📈 Технические Индикаторы
- **RSI (14)** - Индекс относительной силы с цветовой индикацией
  - Зелёный < 30 (перепроданность)
  - Красный > 70 (перекупленность)
- **MACD** - Схождение/расхождение скользящих средних
- **EMA (21)** - Экспоненциальная скользящая средняя
- **ATR** - Средний истинный диапазон (волатильность)

#### 🤖 ML Сигналы
- **BUY** - Сигнал на покупку с уровнем уверенности
- **SELL** - Сигнал на продажу с уровнем уверенности
- **NEUTRAL** - Нейтральный сигнал
- Генерация на основе комбинации индикаторов

#### 💹 Симуляция Торговли
- Размещение ордеров **BUY/SELL**
- Выбор типа ордера (Market / Limit)
- Управление количеством
- Отслеживание открытых позиций
- Расчёт прибыли/убытков (PnL) в реальном времени

#### 🎨 Профессиональный Интерфейс
- Тёмная тема для комфортной работы
- Адаптивный дизайн для всех устройств
- Боковая навигационная панель
- Информационные карточки с ключевыми метриками
- Уведомления о действиях

## 🚀 Как Использовать

### Вариант 1: Онлайн (GitHub Pages)

1. Перейдите по ссылке: **https://mamaspalil.github.io/program/**
2. Страница автоматически загрузит демо
3. Начните исследовать функционал

### Вариант 2: Локально

1. Клонируйте репозиторий:
```bash
git clone https://github.com/MamaSpalil/program.git
cd program/web-version-php/frontend
```

2. Откройте `demo.html` в браузере:
```bash
# Linux
xdg-open demo.html

# macOS
open demo.html

# Windows
start demo.html
```

### Вариант 3: Локальный Сервер

```bash
# С Python
cd web-version-php/frontend
python3 -m http.server 8000
# Откройте: http://localhost:8000/demo.html

# С Node.js
npx http-server -p 8000
# Откройте: http://localhost:8000/demo.html

# С PHP
php -S localhost:8000
# Откройте: http://localhost:8000/demo.html
```

## 🎮 Инструкция по Использованию

### 1. Выбор Символа и Интервала
- Выберите криптовалютную пару из выпадающего списка
- Выберите временной интервал (1m, 5m, 15m, 1h, 4h, 1d)
- График и индикаторы обновятся автоматически

### 2. Просмотр Индикаторов
- **RSI**: Следите за уровнями перекупленности/перепроданности
- **MACD**: Анализируйте тренд и моментум
- **EMA**: Используйте как уровень поддержки/сопротивления
- **ATR**: Оценивайте волатильность рынка

### 3. ML Сигналы
- Смотрите на сигнал в правом верхнем углу
- **BUY (зелёный)**: Рекомендация к покупке
- **SELL (красный)**: Рекомендация к продаже
- **NEUTRAL (жёлтый)**: Нет чёткого сигнала
- Процент показывает уверенность модели

### 4. Размещение Ордеров
- Введите количество в поле "Quantity"
- Выберите тип ордера (Market/Limit)
- Нажмите **BUY** для длинной позиции
- Нажмите **SELL** для короткой позиции
- Позиция появится в списке "Open Positions"

### 5. Управление Позициями
- Просматривайте открытые позиции
- Следите за PnL в реальном времени
- Анализируйте прибыльность сделок

### Особенности Реализации

**Генерация Свечей:**
- Базовая цена + случайная волатильность (0.2%)
- OHLC генерируется реалистично
- 200 исторических свечей

**Расчёт Индикаторов:**
- RSI: 14-периодный расчёт на основе gains/losses
- MACD: Разница между EMA(12) и EMA(26)
- EMA: Экспоненциальная скользящая средняя
- ATR: Средний истинный диапазон за 14 периодов

**ML Сигналы:**
- Комбинация RSI, MACD и положения цены относительно EMA
- Динамический расчёт уверенности (40-95%)
- Логика: RSI < 35 + цена < EMA + MACD < 0 = BUY

## ❓ FAQ

**Q: Это реальные данные?**
A: Нет, все данные симулируются локально в браузере. Для реальных данных используйте полную версию с PHP backend.

**Q: Можно ли потерять реальные деньги?**
A: Нет! Это демо работает с виртуальными данными. Никакие реальные сделки не совершаются.

**Q: Требуется ли интернет?**
A: Только для загрузки библиотеки TradingView Charts (CDN). После загрузки всё работает локально.

**Q: Как установить полную версию?**
A: См. основной [README.md](README.md) для инструкций по установке PHP backend, Java ML service и MSSQL базы данных.

**Q: Поддерживаются ли другие биржи?**
A: В демо - нет. Полная версия поддерживает Binance, Bybit, OKX, Bitget, KuCoin.

**Q: Можно ли добавить свои индикаторы?**
A: Да! Откройте `assets/js/demo.js` и добавьте функции расчёта новых индикаторов.
